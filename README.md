## TaskNotifier_BeefPlus について
- 2025/11/11 - v1.0をリリース

この拡張機能は、神戸大学の学習支援システム BEEF+ から課題一覧を取得し、締切が近づいたタイミングでブラウザ通知を送る Chrome 拡張機能です。課題の取りこぼしを防ぎたいときに、ブラウザから手軽に残り時間をチェックできます。

GitHub で公開している個人プロジェクトであり、神戸大学や BEEF+ 運営とは一切関係がありません。利用やセットアップに伴う不具合・情報漏えいなどのリスクは各自の責任でご判断ください。本リポジトリのコードを利用したことによって生じたいかなる損害についても、開発者は責任を負いません。

導入手順や変更点は今後 README に追記予定です。拡張機能の挙動や改善アイデアがあれば Issue や Pull Request で気軽に共有してください。

## Chrome へのインストール手順

1. このリポジトリを GitHub で開き、`Code` → `Download ZIP` から最新コードを取得します。
2. ダウンロードした ZIP を任意のフォルダに展開します。
3. Chrome で `chrome://extensions/` を開き、右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、手順 2 で展開したフォルダを選択します。
5. 拡張機能が追加されたら、通知の許可設定などを必要に応じて確認してください。
